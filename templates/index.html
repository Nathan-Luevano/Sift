About Sift

Sift is a local project for comparing forensic timeline data with public reporting, social posts, and web results.

  • Parse E01 and raw-style evidence images
  • Pull in Reddit, Twitter/X, news, and web results
  • Score timeline and location overlaps
  • Review the results in one place
Quick Start
  1. Create an investigation
  2. Upload one or more evidence files
  3. Collect OSINT for the same place and time window
  4. Run the correlation pass
  5. Review the timeline, map, and analytics views
